# 📖 Project Overview

Smart Build Hub is a full-stack web application developed using the Django framework (Python) to simplify home construction management.
The platform connects Homeowners, Contractors, and Administrators through a centralized system where users can explore construction services, manage projects, monitor progress, and communicate efficiently.
The primary goal of this project is to make home construction more transparent, affordable, and organized by providing budget-friendly solutions and trusted contractors for families ranging from middle-income to premium homeowners.

 # 📊 Project Highlights

- Built using Django (Python Framework)
- Implemented Role-Based Authentication
- Designed Responsive User Interface using Bootstrap
- Followed MVC (MVT in Django) Architecture
- Managed Data using SQLite Database
- Version Controlled using Git & GitHub
